# GitHub wiki style customizer
GitHubのwikiのスタイルを変えられたらいいなぁと思いました。しらべたところ、GitHubの公式機能としては備わってないみたいです。どこかにスタイルシートをうｐしてそれを使ってもらうこともできないらしいです。

# そこで
自鯖上での動作しかできませんが、wikiの本体部分でhtml化された部分を取得する関数を作りました。こんなもん生粋のPHPerさんは一瞬で書けるコードですが自分は反日かかりました。

# どうやって使うの
getWikiText.phpをrequireし、getWikiText関数を呼んでください。引数に、`user_name`・`repository_name`・`page_id`の三つを指定して呼んでください。Wiki本体（下記）のデータのみ拾ってきます。

```
...
<div class="markdown-body">
...(wiki本体)...
</div>
...
```

ただ、今のところ開始を363行目からと決め打ちしているので、後々バグるかもしれません。
また、htmlspecialcharsだっけかを挟んでいないので、githubのwikiがやらかした場合は一緒にやらかします。githubのwikiが既にそういう対策がされているという前提の他力本願なやつです。

ユーザー名とリポ名は空白だと、返される文字列も空白です。また取得できる文字がなかった場合とかも空白が返ります。
